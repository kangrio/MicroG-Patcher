# MicroG Patcher

## usage
* Download and Install from this repo [GMS Core](https://github.com/kangrio/GmsCore-User/releases)
    * **microG Services**
    * **microG Companion**
    * **Services Framework Proxy**
* Download and Install this app
* Open this app
* Click select apk
* Choose apk to patch and wait for finish
* Install patched apk
* Done
